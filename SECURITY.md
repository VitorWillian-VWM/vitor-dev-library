# 🔒 Security Policy

Obrigado por ajudar a manter a **Open Computer Science Library** segura.

Este documento descreve como reportar vulnerabilidades e quais versões do projeto recebem suporte.

---

# 📦 Versões Suportadas

| Versão | Suporte |
|---------|----------|
| Main | ✅ Sim |
| Releases | ✅ Sim |
| Versões antigas | ❌ Não |

Sempre recomendamos utilizar a versão mais recente do repositório.

---

# 🛡️ Reportando uma Vulnerabilidade

Se você encontrar algum problema relacionado à segurança do projeto, por favor **não abra uma Issue pública**.

Em vez disso, entre em contato diretamente através do GitHub ou envie uma mensagem privada ao mantenedor do projeto.

Ao reportar uma vulnerabilidade, informe:

- descrição detalhada;
- impacto esperado;
- como reproduzir;
- arquivos afetados;
- possíveis sugestões de correção.

---

# 🔐 Escopo

Esta política cobre:

- GitHub Actions
- Templates
- Scripts
- Documentações
- Estrutura do repositório
- Site oficial (quando disponível)

Caso futuramente existam aplicações web vinculadas ao projeto, elas também passarão a fazer parte desta política.

---

# 📚 Conteúdo dos Livros

Os livros deste repositório têm finalidade educacional.

Caso seja encontrado:

- código inseguro;
- exemplos vulneráveis;
- informações desatualizadas;
- práticas consideradas inadequadas;

abra uma Issue ou envie um Pull Request propondo a correção.

Nosso objetivo é manter todo o conteúdo técnico atualizado de acordo com as boas práticas da comunidade.

---

# 🚫 Divulgação Responsável

Pedimos que qualquer vulnerabilidade seja comunicada de forma responsável.

Evite divulgar publicamente falhas antes que exista tempo suficiente para análise e correção.

---

# 🤝 Nosso Compromisso

Sempre que um problema de segurança for reportado, faremos o possível para:

- confirmar o problema;
- analisar seu impacto;
- implementar uma correção;
- atualizar a documentação quando necessário;
- agradecer publicamente ao responsável (quando autorizado).

---

# ❤️ Obrigado

A segurança também faz parte da educação.

Obrigado por contribuir para tornar a **Open Computer Science Library** um projeto cada vez mais confiável para estudantes, professores e toda a comunidade Open Source.
