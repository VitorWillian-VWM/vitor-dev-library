# 📖 Sobre o Livro

**Linguagem de Programação** é um livro introdutório desenvolvido para apresentar os principais conceitos que fundamentam a programação moderna.

Ao contrário de muitos materiais que começam ensinando diretamente uma linguagem específica, esta obra busca explicar primeiro como ocorre a comunicação entre o ser humano e o computador, permitindo que o leitor compreenda a base da Ciência da Computação antes de escrever seus primeiros programas.

Durante a leitura são apresentados conceitos fundamentais como linguagem de máquina, linguagens de baixo nível, Assembly, Linguagem C, compiladores, interpretadores, além de introduções à Arquitetura de Computadores, Organização de Computadores, Engenharia de Software e conceitos matemáticos aplicados à programação.

O objetivo principal é construir uma base sólida de conhecimento para facilitar o aprendizado de qualquer linguagem de programação no futuro.

---

# 🎯 Objetivos

- Explicar os fundamentos da programação;
- Demonstrar como um computador interpreta um programa;
- Introduzir conceitos de Arquitetura de Computadores;
- Apresentar a evolução das linguagens de programação;
- Introduzir a Linguagem Assembly;
- Explicar a importância da Linguagem C;
- Demonstrar como funcionam compiladores e interpretadores;
- Apresentar conceitos matemáticos utilizados na computação;
- Preparar o estudante para aprender qualquer linguagem moderna.
