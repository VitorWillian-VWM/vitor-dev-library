# 🤝 Guia para Contribuidores

Obrigado por considerar contribuir com a **Open Computer Science Library**.

Este projeto nasceu com o objetivo de disponibilizar gratuitamente livros, apostilas e materiais de estudo sobre Ciência da Computação, Programação, Engenharia de Software, Inteligência Artificial e diversas outras áreas da tecnologia.

Toda contribuição é bem-vinda.

---

# 🎯 Como você pode contribuir

Você pode ajudar o projeto de diversas maneiras:

- 📚 Escrevendo novos livros;
- ✍ Corrigindo erros de português;
- 💡 Sugerindo novos conteúdos;
- 🐛 Reportando erros;
- 🔧 Corrigindo problemas encontrados;
- 🌎 Traduzindo materiais;
- 📖 Melhorando explicações;
- 🎨 Melhorando imagens e diagramas;
- 💻 Criando exemplos de código;
- 📝 Melhorando a documentação.

---

# 📂 Estrutura dos Livros

Cada livro possui sua própria pasta dentro do diretório:

```text
books/
├── linguagem-de-programacao/
├── inteligencia-artificial-5-anos/
├── logica-de-programacao/
└── ...
```

Cada livro normalmente contém:

```text
📦 nome-do-livro
├── imagens/
├── slide/
├── source/
├── README.md
└── livro.pdf
```

---

# 📝 Padrão dos Livros

Sempre que possível, procure manter o seguinte padrão:

- linguagem simples;
- conteúdo didático;
- exemplos práticos;
- imagens ilustrativas;
- capítulos organizados;
- referências utilizadas;
- README atualizado.

---

# 📚 Fontes de Pesquisa

Sempre utilize fontes confiáveis.

Exemplos:

- Documentações oficiais
- Livros
- Artigos científicos
- Universidades
- Papers
- RFCs
- Documentação técnica

Quando utilizar conteúdos externos, cite a referência.

---

# 🚀 Como contribuir

1. Faça um Fork do projeto.

2. Crie uma Branch.

```bash
git checkout -b feature/meu-livro
```

3. Faça suas alterações.

4. Realize o commit.

```bash
git commit -m "docs: adiciona capítulo sobre estruturas de dados"
```

5. Envie sua Branch.

```bash
git push origin feature/meu-livro
```

6. Abra um Pull Request.

---

# ✔ Antes de enviar um Pull Request

Verifique se:

- O texto está revisado;
- Não existem erros de ortografia;
- Os exemplos funcionam corretamente;
- As imagens possuem boa qualidade;
- Os arquivos estão organizados;
- O README foi atualizado, se necessário.

---

# 💬 Sugestões

Caso não queira escrever código, você ainda pode contribuir:

- Abrindo uma Issue;
- Sugerindo melhorias;
- Indicando novos livros;
- Corrigindo informações técnicas;
- Encontrando erros de digitação.

---

# ❤️ Nosso objetivo

A **Open Computer Science Library** pretende se tornar uma biblioteca aberta de materiais gratuitos para estudantes e profissionais da área de tecnologia.

Cada contribuição ajuda a tornar o conteúdo mais completo e acessível para toda a comunidade.

Obrigado por fazer parte deste projeto! 🚀
