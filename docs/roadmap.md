# 🗺️ Roadmap

Este documento apresenta o planejamento da **Open Computer Science Library**.

O objetivo deste roadmap é mostrar a evolução esperada da biblioteca e os próximos conteúdos que serão desenvolvidos.

> **Importante:** este planejamento poderá ser atualizado conforme novos livros, tecnologias e ideias surgirem durante o desenvolvimento do projeto.

---

# 🎯 Missão

Construir uma biblioteca Open Source de livros gratuitos sobre Ciência da Computação, Programação, Engenharia de Software e Inteligência Artificial, oferecendo materiais didáticos, acessíveis e constantemente atualizados.

# 🚀 Visão de Longo Prazo

A Open Computer Science Library pretende evoluir para uma biblioteca aberta de referência para estudantes, professores e profissionais da área de tecnologia.

O projeto busca reunir conteúdos que acompanhem a evolução da Computação, oferecendo materiais organizados, gratuitos e de qualidade para pessoas de diferentes níveis de conhecimento.

Este roadmap será revisado periodicamente conforme novos livros forem publicados e novas tecnologias surgirem.
