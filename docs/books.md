# 📚 Catálogo de Livros

Bem-vindo à biblioteca da **Open Computer Science Library**.

Nesta página você encontrará todos os livros publicados e aqueles que estão em desenvolvimento.

O objetivo deste projeto é disponibilizar gratuitamente materiais de estudo sobre Ciência da Computação, Programação, Engenharia de Software, Inteligência Artificial e diversas outras áreas da tecnologia.

---

# 📖 Livros Publicados

## 📘 Linguagem de Programação

**Status:** ✅ Disponível

Uma introdução ao funcionamento da programação desde seus níveis mais baixos até as linguagens modernas.

### Conteúdo

- História das linguagens
- Linguagem de Máquina
- Código Binário
- Assembly
- Linguagens Compiladas
- Linguagens Interpretadas
- Linguagens Híbridas
- Exemplos em diversas linguagens
- Glossário

📂 Pasta

```text
books/
└── linguagem-de-programacao/
```

# 🎯 Objetivo da Biblioteca

Todos os livros deste projeto possuem o mesmo objetivo:

- 📚 Compartilhar conhecimento gratuitamente.
- 👨‍🏫 Auxiliar estudantes.
- 💻 Apoiar desenvolvedores iniciantes.
- 🌎 Incentivar a comunidade Open Source.
- 🚀 Tornar conteúdos técnicos mais acessíveis.
